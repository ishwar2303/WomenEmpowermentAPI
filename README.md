# WomenEmpowermentAPI

## STEP Controllers

### TraineeController
```
Route: api/Trainee/Register
Method: POST
Action: PostRegister
```

```
Route: api/Trainee/Login
Method: POST
Action: PostLogin
```

Trainee Registration and Login are performed via this controller.
